IDE

Un entorno de desarrollo integrado (IDE) es un sistema de software de diseño de aplicaciones que combina herramientas de desarrollo comunes en una única interfaz gráfica de usuario (GUI). En general, los IDE tienen las siguientes características:

  1. Editor de código fuente: un editor de texto que lo ayuda a escribir código de programa con funciones como resaltado de sintaxis con señales visuales, autocompletado para el idioma seleccionado y verificación de errores al escribir código.
  2. Automatización de compilación local: herramientas que automatizan tareas simples y repetitivas como parte de la creación de una arquitectura de software local para que la usen los desarrolladores, como compilar una fuente de computadora en código binario, encapsular ese código y ejecutar pruebas automatizadas.
  3. Depurador: un programa utilizado para probar otros programas y mostrar gráficamente la ubicación de los errores en el código original.
